Title: Interface, HUD & Operator Controls

Objective:
- Provide an operator-facing HUD layer (OLED + terminal dashboard + optional web status panel).
- Expose system state, network mode, logging state, and running tools at a glance.

Key Decisions:
- Script-first data sources (bash scripts output KEY=VALUE)
- Render targets: OLED (optional), tmux/CLI dashboard, web mini status panel (optional)

Operator UX Principles:
- Low cognitive load
- High signal / low noise
- Text-first
- Fail-safe indicators for mode + capture/logging state
